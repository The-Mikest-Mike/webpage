## 📡 API Usage
- **Endpoint:** `POST https://tmmsoftware-resend.vercel.app/api/send-email`
- **Request:**
```json
{
  "email": "user@example.com",
  "tosAccepted": true
}
```
- **Response:**
```json
{
  "message": "Email sent successfully!"
}
```
